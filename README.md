# Supervised Learning - (Machine Learning)

Supervised Learning is a type of machine learning where a model is trained using labeled data, meaning each input has a corresponding known output. The algorithm learns the relationship between input features and the target variable by analyzing example inputâ€“output pairs. Once trained, the model can predict outcomes for new, unseen data.

Supervised learning is mainly used for classification (predicting categories such as spam or not spam) and regression (predicting continuous values such as prices or scores). It is widely applied in real-world problems like disease prediction, fraud detection, sales forecasting, and recommendation systems because of its accuracy and interpretability.
